Academic Projects

Bird Call Classification

Oct 2023 - Nov 2023

I built a deep learning model to classify bird calls using a smaller subset of the BirdCLEF 2023 dataset. The project used audio preprocessing techniques like noise reduction and peak-based cropping to create clear Mel Spectrograms. With an EfficientNet model and GeM pooling, it achieved an accuracy of 78%.

Cube Dodger

Sept 2023 - Nov 2023

I developed an endless runner game from scratch using Unity Engine and C#. All aspects — including animations, game logic, UI design, and custom assets — were created from first principles. I also composed original music and sound effects using Bosca Ceoil. The game features smooth animations, dynamic obstacle spawning, and accurate collision detection, and was later deployed using WebGL.
